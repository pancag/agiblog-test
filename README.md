# Teste Técnico QA - Blog do Agi

## Descrição
Este repositório contém testes automatizados para a pesquisa no Blog do Agi.

## Pré-requisitos
- Python 3.x
- Selenium
- WebDriver para o seu navegador (ex: ChromeDriver)

1. Clone este repositório:
   ```bash
   git clone https://github.com/SEU_USUÁRIO/teste-qa-agiblog.git
   cd teste-qa-agiblog

2. Instale as dependências:
   ```pip install selenium

3. Execucão dos testes
